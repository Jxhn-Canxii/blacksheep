## Development Workflow

1. `npm install`
2. `npm run dev`
3. Access at `http://localhost:3000`

## Tech Stack
- **Framework**: Next.js 16 (App Router)
- **Database**: Supabase
- **Real-time**: Supabase Realtime
- **Animations**: Framer Motion
- **Map**: Leaflet / React Leaflet
- **Icons**: React Icons (Hi, Bi, Ri, Md, Rx)
