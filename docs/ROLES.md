# Role Responsibilities

This document outlines the roles and responsibilities within the Black Sheep project.

## Core Team Roles

- **Product Owner**: Responsible for defining project goals, prioritizing the roadmap, and ensuring the product meets user needs.
- **Lead Developer**: Responsible for technical architecture, code quality standards, and major implementation decisions.
- **Frontend Engineer**: Focuses on UI/UX implementation, responsive design, and client-side logic using Next.js and Tailwind CSS.
- **Backend/Supabase Engineer**: Manages database schema, Row Level Security (RLS) policies, and server-side logic/API routes.
- **UI/UX Designer**: Responsible for the visual aesthetic, brand identity (Baara mascot), and overall user experience.

## Project Management

- **Roadmap Management**: Handled via `TODO.md` (to be recreated) and project boards.
- **Documentation**: All team members are responsible for maintaining up-to-date documentation in the `docs/` directory.
- **Audit & Compliance**: Periodic reviews for coding standards, accessibility, and security.
